## 刷题建议

1. 按专题刷:每专题先做代表题建立套路,再横向扩展变式。
2. 高频题(★★★★★)优先,见 [高频算法题 Top 40](../indexes/高频题目索引.md#a-高频算法题-top-40)。
3. 每题留 Java 解 + 复杂度 + 一句变式;难题写专题小结。
4. 推荐顺序:数组/链表/二叉树 → 双指针/滑窗/二分 → 回溯/动态规划 → 图/并查集/单调栈 → 设计题。
