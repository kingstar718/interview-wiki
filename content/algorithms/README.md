## 刷题建议

1. 按套路刷:每个套路先做代表题建立模式,再横向扩展变式——套路页的"为什么这些题是同一个套路"就是这条主线。
2. 高频题(★★★★★)优先,见 [高频算法题 Top 40](高频题目索引.md#a-高频算法题-top-40)。
3. 每题留 Java 解 + 复杂度 + 一句变式;难题写关联题互链。
4. 推荐顺序:数组/链表/二叉树 → 双指针滑窗/二分/哈希表 → 回溯/动态规划 → 图论/栈与队列 → 排序与堆/字典树/位运算。
