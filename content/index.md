## 怎么用

1. **刷算法**:从 [算法题索引](indexes/01-算法题索引.md) 选专题 → 按频次 ★★★ 优先 → 详解写进 `algorithms/<专题>/`。
2. **背八股**:从 [社招问题知识点](社招问题知识点.md) 进个人枢纽,按 `[[双链]]` 跳专题,页底看反向链接。
3. **冲刺大厂**:从 [高频题目索引](indexes/03-高频题目索引.md) 自上而下刷高频,按目标公司重点准备。

## 详解模板

- **算法题**:思路 → 复杂度 → Java 代码 → 边界/变式 → 易错点
- **知识点**:是什么 → 为什么这么设计 → 对比同类 → 常见追问 → 代码/图示

## 约定

- 频次:★★★★★ 必考 / ★★★★ 高频 / ★★★ 常见 / ★★ 偶考
