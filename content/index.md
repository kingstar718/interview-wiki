## 怎么用

1. **背八股**:左侧「面试专题」按学习顺序排列(Java → 框架 → 数据库 → 中间件 → 计算机基础 → 分布式与架构 → 工程实践 → 面试),每篇顶部有「面试追问地图」,先看地图再进正文。
2. **刷算法**:从 [算法题索引](算法题索引.md) 选套路 → 套路页讲「为什么这些题是同一类」,技术词讲「具体用哪一招」;题解末尾的「变式/关联题」是串题的线索。
3. **冲刺大厂**:从 [高频题目索引](高频题目索引.md) 自上而下刷高频,按目标公司重点准备。

## 内容体例

- **知识点小节**:是什么 → 为什么这么设计 → 源码 → 对比同类 → 常见追问 → 通用概念
- **算法题解**:题目 → 思路 → 代码 → 复杂度 → 边界条件 → 变式 → 易错点 → 面试追问 → 关联题
- **元数据行**:`频次 ★★★★ · 难度 🟡 · 高频:字节/美团`
  - 频次:★★★★★ 必考 / ★★★★ 高频 / ★★★ 常见 / ★★ 偶考
  - 难度:🟢 易 / 🟡 中 / 🔴 难
