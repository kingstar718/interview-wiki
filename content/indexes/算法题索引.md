# 算法题索引

> 按「套路」组织(非按数据结构分类)——同一套路归一类,便于举一反三。一题可能横跨多个套路(如「接雨水」同时是数组、双指针、单调栈的例题),各套路页的详解是同一份物理题解,不是重复内容。
> 频次:★★★★★ 几乎必考 / ★★★★ 高频 / ★★★ 常见 / ★★ 偶考。难度:🟢易 🟡中 🔴难。
> 公司标注为大致倾向,以实际面试为准:[阿里][腾讯][字节][美团][百度][京东][拼多多][滴滴][网易][快手]。
> 题解都在 `algorithms/problems/`,套路页(`algorithms/<套路名>.md`)是分类视图 + 为什么该套路成立的讲解,模板:思路 → 复杂度 → Java 代码 → 边界/变式 → 易错点。
> 相关:[高频题目索引](高频题目索引.md)(按热度)。
